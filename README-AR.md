# 🏦 نظام إدارة البنك - المشروع الثالث (الإصدار الأول) 🚀
[ **🇺🇸 Read in English** ](./README.md)

![C++](https://img.shields.io/badge/Solutions-C++-blue.svg) ![Status](https://img.shields.io/badge/Status-Completed-success.svg)

تطبيق كونسول احترافي بلغة ++C لإدارة عملاء البنك. يركز هذا المشروع على تطبيق العمليات المصرفية الأساسية مع حفظ البيانات بشكل دائم باستخدام الملفات.

## 🌟 ميزات الإصدار الأول:
* 📊 **عرض قائمة العملاء:** عرض جميع العملاء في جدول منظم واحترافي.
* ➕ **إضافة عميل جديد:** إضافة سجلات جديدة مع التحقق من عدم تكرار رقم الحساب.
* ❌ **حذف عميل:** نظام حذف آمن يتطلب التأكيد قبل المسح النهائي.
* 🔄 **تحديث بيانات العميل:** تعديل (الاسم، الهاتف، الرمز السري، الرصيد) بسهولة.
* 🔍 **البحث عن عميل:** ميزة البحث السريع باستخدام رقم الحساب.

## 🛠️ الجوانب التقنية:
* 💻 **اللغة:** C++.
* 📁 **التخزين:** استخدام ملفات نصية (`Clients.txt`) لضمان بقاء البيانات.
* 🏗️ **هياكل البيانات:** استخدام `struct` لتمثيل البيانات و `vector` لإدارتها في الذاكرة.
* ⚙️ **المنطق البرمجي:** بناء خوارزمية `SplitString` مخصصة لمعالجة النصوص.
