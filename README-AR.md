# 🏦 نظام إدارة البنك - المشروع الثالث (الإصدار الثاني) 🚀
[ **🇺🇸 Read in English** ](./README.md)

![C++](https://img.shields.io/badge/Solutions-C++-blue.svg) ![Status](https://img.shields.io/badge/Status-Completed-success.svg)

هذا هو **التطوير الأول (Extension 1)** لمشروع نظام البنك. في هذا الإصدار، قمت بتحويل البرنامج إلى نظام بنكي حقيقي عبر إضافة **نظام المعاملات** لإدارة أموال العملاء بشكل آمن.

## 🌟 الإضافات الجديدة في الإصدار الثاني:
* 💸 **قائمة المعاملات:** قائمة فرعية جديدة مخصصة بالكامل للعمليات المصرفية.
* 📥 **الإيداع:** إمكانية إضافة مبالغ لحساب أي عميل مع تحديث فوري للملفات.
* 📤 **السحب:** نظام سحب آمن يتضمن التحقق من الرصيد المتاح قبل إتمام العملية.
* 📊 **إجمالي الأرصدة:** تقرير شامل يعرض أرصدة جميع العملاء ومجموع الأموال الكلي في البنك.
* 🔄 **تطوير التنقل:** سلاسة تامة في التنقل بين القائمة الرئيسية وقائمة المعاملات.

## 🛠️ التحسينات التقنية:
* ⚙️ **منطق القوائم الفرعية:** استخدام `enum` و `switch cases` لتنظيم تدفق البرنامج.
* ✅ **التحقق من البيانات:** إضافة منطق برمجي يمنع سحب مبالغ أكبر من الرصيد المتاح.
* 💾 **التحديث الديناميكي:** يقوم النظام بإعادة كتابة الملف فور نجاح أي معاملة لضمان دقة البيانات.
